---
title: Send Image
api: POST /v1/messages/send/image
description: "Send an image to a WhatsApp number"
---

Send an image with optional caption.

## Request Body

<ParamField body="phone" type="string" required>
  Recipient phone number in international format
</ParamField>

<ParamField body="image" type="string" required>
  Image URL or base64-encoded data in data URI format (`data:image/jpeg;base64,...`)
</ParamField>

<ParamField body="caption" type="string">
  Optional caption for the image
</ParamField>

<ParamField body="id" type="string">
  Optional message ID to reply to
</ParamField>

## Supported Formats

- JPEG / JPG
- PNG
- GIF
- WEBP

<Note>Maximum file size: 16MB</Note>

<RequestExample>
```bash cURL
curl -X POST https://api.livchat.ai/v1/messages/send/image \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "phone": "5511999999999",
    "image": "https://example.com/image.jpg",
    "caption": "Check this out!"
  }'
```

```javascript JavaScript
const response = await fetch("https://api.livchat.ai/v1/messages/send/image", {
```

```python Python
import requests

    "https://api.livchat.ai/v1/messages/send/image",
    headers={

    json={

    },
)
```
