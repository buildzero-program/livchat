---
title: Disconnect Session
api: POST /v1/session/disconnect
description: "Disconnect from WhatsApp servers"
---

Disconnect from WhatsApp servers while keeping the session active. Reconnecting later won't require a new QR scan.

<Note>
Use this when you need to temporarily stop receiving messages. The session remains valid for reconnection.
</Note>

<RequestExample>
```bash cURL
curl -X POST https://api.livchat.ai/v1/session/disconnect \
  -H "Authorization: Bearer YOUR_API_KEY"
```

```javascript JavaScript
const response = await fetch("https://api.livchat.ai/v1/session/disconnect", {
  method: "POST",
  headers: {
    Authorization: "Bearer YOUR_API_KEY",
  },
});
```
</RequestExample>

<ResponseExample>
```json 200
{
  "success": true,
  "code": 200,
  "data": {
    "details": "Disconnected"
  }
}
```
</ResponseExample>
