## 3. Features Enterprise

Arquivos e diretórios identificados como Features Enterprise estão
sujeitos a restrições adicionais.

### Identificação

Features Enterprise são identificadas por:

- Arquivos contendo `.ee.` no nome (ex: `white-label.ee.ts`)
- Diretórios nomeados `/ee/` (ex: `src/features/ee/`)

### Restrições

- O **código é visível** para fins educacionais
- O **uso em produção** requer assinatura ativa do LivChat Cloud
- Membros BuildZero e público geral **NÃO podem usar** Features
  Enterprise sem assinatura

### Exemplos de Features Enterprise

- White Label (remoção de branding)
- Multi-tenant avançado
- Analytics premium
- Integrações enterprise
